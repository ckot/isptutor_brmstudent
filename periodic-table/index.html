Periodic Table

This periodic table shows the different types of atoms, or elements, that we know about. All elements have some weight--even the lightest element, hydrogen (H), has some weight. The weights of each element (in "atomic mass units") are shown in the upper left corner of each box.Â 

The different elements also have different numbers of PROTONS. The number of protons for each element is shown in the upper RIGHT corner of each box. This is called the atomic number. For example Hydrogen (H) only has one proton. Helium (He) has 2 protons, etc. The number in the upper right corner of each box is also the number of ELECTRONS that element has (when the element has no net electric charge--or has the same number of protons and electrons).
