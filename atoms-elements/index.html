Atoms-Elements

Elements. There are many different types of atoms, called elements. Hydrogen, carbon, oxygen, sodium, and chlorine are a few different elements. An individual atom can be classified as a specific element based on the number of protons the atom has. 

The figure below is a drawing of two different elements: hydrogen and carbon. Hydrogen is the simplest element, with just one proton and one electron (and no neutrons!). Oxygen is more complex (with 8 protons). Your body is made of a whole lot of atoms of these two elements! 



Atoms in their basic "elemental" forms have no "net" charge. That is, they have the same number of electrons and protons.

The elements we have discovered (so far!) are shown in Periodic Tables like this one. Elements in Periodic tables are ordered by "atomic number," which is the number of protons the element has. The atomic number/number of protons is shown in the top right corner of each box for each element. For example, as you can see in the Periodic Table, the element Sodium (Na) has 11 protons.

Ions: Sometimes elements gain or lose one or more electrons. Because the atoms no longer have the same number of protons (+) and electrons (-), they have a charge. Elements with additional or fewer electrons are called ions. 

  • One example of an ion is a Hydrogen (H) atom that has lost an electron. It now has one more proton (+) than electron (-), so it has a positive charge. A Hydrogen ion is written as H+. 
  • The "+" sign indicates that it has a "net" positive charge, or there are more (positively-charged) protons than (negatively-charged) electrons.
  • Another example of an ion is a Fluorine (F) atom that has gained an electron. It now has one more electron than proton, so it has a negative charge. A Fluorine ion (called Fluoride) is written as Fl-.  
  • The "-" sign indicates it has a "net" negative charge, or there are more (negatively-charged) electrons than (positively-charged) protons.
